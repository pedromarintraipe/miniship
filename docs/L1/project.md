# Worship App

## Descripción

Aplicación web Full-Stack para ministerios de alabanza que permite gestionar canciones, importar letras con acordes, transponer tonos y crear setlists.

## Objetivo

Facilitar ensayos y presentaciones en vivo mediante una herramienta rápida, simple y adaptada al contexto real.

## Enfoque

- Uso en vivo
- Simplicidad
- Sin sobreingeniería
